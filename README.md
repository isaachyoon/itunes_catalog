# itunes_catalog

![](itunes_gif.gif)

## To start the project..

### Client: go to /itunes_catalog/client and run `npm start`
### Server: Run `npm start`to start the Server
### Mongo : Download Mongo and run `Mongod` to run a database locally

## Tech Stack

### Front-End: React + Redux
### Server: Node / Express
### Database: Mongo
